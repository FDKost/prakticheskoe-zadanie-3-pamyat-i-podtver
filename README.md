# Практическое задание №3: Память и подтверждение действий

## Requirements
- [high] Установить rich: Добавить зависимость pip install rich и использовать Console для вывода сообщений.
- [high] Добавить память разговоров: Использовать MemorySaver и передавать thread_id в config, чтобы агент запоминал историю сообщений.
- [high] Включить паузу перед инструментом: Настроить interrupt_before=['tools'] и убедиться, что checkpointer установлен, чтобы агент останавливался перед каждым вызовом инструмента.
- [high] Обработать паузу в стриме: Внутри цикла обработки chunk'ов проверять наличие ключа '__interrupt__' и state.next == ('tools',) для запроса подтверждения у пользователя.
- [high] Реализовать подтверждение и возобновление: При подтверждении вызвать ask_and_run(None, config) для возобновления, иначе вывести сообщение об отмене и выйти из цикла.
- [high] Создать функцию ask_and_run: Объединить логику стриминга, вывода токенов, вызовов инструментов и обработки паузы в одну функцию.
- [high] Разработать чат‑цикл: В основном цикле принимать ввод пользователя, вызывать ask_and_run, обрабатывать команду exit.
